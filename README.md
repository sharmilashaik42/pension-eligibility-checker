# Mini Project Report  
## Pension Eligibility Checker  

### Project Title:
**Pension Eligibility Checker**

### Overview:
This mini project is a web-based tool developed using **HTML**, **CSS**, and **JavaScript**. It allows users to enter their age and determines their eligibility for pension benefits based on predefined age brackets.

### Features:
- Takes user input (age) and validates it  
- Age-based pension eligibility messages  
- Modern, user-friendly interface with responsive design  
- No backend required – fully client-side  
- Built-in styling using CSS and Google Fonts  

### Age Categories:
- **Below 18**: No pension  
- **21–30**: ₹5000 pension  
- **31–40**: ₹6000 pension  
- **41–49**: ₹7000 pension  
- **51–59**: ₹8000 pension  
- **60 and above**: ₹10000 pension
